
### 🌦️ Weather / Météo

When asked about "meteo" or "météo":
- If no location is specified, default to **Angers, France**.
- If a location is specified (e.g., "meteo Rome"), use that location.

**Provide:**
1. **Current conditions**
2. **Today's high/low**
3. **7-day forecast summary**
4. **Any extreme weather warnings**

**Style:** Brief. No hourly breakdowns unless requested.
**Proactive:** Warn about unusual events (storms, heat waves, extreme cold) even if not asked.
