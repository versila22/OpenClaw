# HEARTBEAT.md

## Memory Maintenance (every 3 days)
- Read recent memory/YYYY-MM-DD.md files (last 3 days)
- Update MEMORY.md with significant learnings, remove outdated info
- Check if TOOLS.md needs updates

## Skill Self-Audit (weekly)
- Check installed skills are up to date
- Review any tool failures from recent sessions

## Post-Session Learning
- Encode errors and wins in daily notes
- If a pattern repeats 3+ times, add it to MEMORY.md

## Tracking
Check memory/heartbeat-state.json for last check timestamps. Update after each check.
Only run checks that are due (memory maintenance: every 3 days, skill audit: weekly).
