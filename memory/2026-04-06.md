## Leçons apprises
- passlib + bcrypt incompatible sur Python 3.12/3.13 → utiliser pbkdf2_sha256
- Lovable ne redéploie pas depuis GitHub → servir le front soi-même via Docker
- VITE_API_URL dans Dockerfile + code qui ajoute /api/v1 = double prefix → ne pas passer VITE_API_URL au build Docker si le code le gère déjà
- .dockerignore indispensable (dist/ + node_modules/) pour builds propres
- SQLite async + aiosqlite : database lock sur tests si middleware écrit en background → désactiver ActivityTrackerMiddleware dans conftest
- Toujours `git pull` avant deploy VPS, et `docker rmi` si le cache Docker est suspect
