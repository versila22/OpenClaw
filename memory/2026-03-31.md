#### Leçons Railway
- Railway Railpack ne supporte PAS le builder Dockerfile via API (enum: HEROKU, NIXPACKS, PAKETO, RAILPACK)
- `.python-version` ignoré par Railpack
- `serviceInstanceRedeploy` relance le dernier build, pas le dernier commit — il faut delete + recreate le service pour forcer
- Toujours utiliser des versions récentes des packages pour compatibilité Railpack/Python latest
