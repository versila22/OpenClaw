## LIMA App — Lancement projet

### Contexte
- App web pour la Ligue d'Improvisation du Maine-et-Loire (LIMA)
- ~60 adhérents, bureau + CA = admins
- Production réelle + portfolio
- Front existant sur Lovable : https://improv-cabaret-planner.lovable.app/
- Lovable project ID : b1e47b07-b17c-44c8-bbda-cecbf2a6f9d2

### Données récupérées depuis Google Drive
- CSV joueurs (39 joueurs, HelloAsso cotisation)
- CSV adhérents (55 adhérents, HelloAsso adhésion)
- PDF alignement T3 2024-25 (grille joueurs × événements)
- Excel calendrier saison 2025-2026
- PDF trombinoscope 2025-2026 (50 membres)

### Décisions V0
- Stack : FastAPI + PostgreSQL + Railway
- Auth : email/mdp + JWT + activation par token
- 2 rôles app : admin (CA+bureau) / member
- 4 statuts joueur : M (Match), C (Cabaret), L (Loisir), A (Adhérent)
- Loisirs peuvent être DJ ou MJ-MC dans alignements
- Coachs = rôle asso à part (coach entraînements M/C, MJ-MC/AR en spectacle)
- Welsh, Giffard, Germoir, Digital Village = lieux de cabaret
- Affectation admin via grilles d'alignement trimestrielles
- Plan de spectacle : persistance backend (remplace localStorage)
- Historique : saison en cours uniquement (sept → août)
- Pas de LLM en V0
- Import CSV HelloAsso + Import Excel calendrier
